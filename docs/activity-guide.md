# 🎄 圣诞节互换礼物·活动指南

欢迎参加我们的圣诞互换礼物活动（Secret Santa）！为了让大家玩得开心、收发礼物方便，同时保护大家的隐私，我们制作了一个不需要注册账号的小工具。

以下是本次活动的操作指南和隐私说明。

---

## 🧑‍✈️ 如果你是房主（发起人）

你的任务是组局和维持秩序。

1. **创建活动**：填写活动名称和截止时间，系统会生成两把“钥匙”：
    * 🔑 **管理 Key**：**请务必私密保存！** 这是你管理后台的唯一凭证。
    * 📨 **邀请 Key**：把它发到群里，邀请朋友们参加。
2. **管理成员**：
    * 使用`管理 Key`登录，你可以看到谁报名了（只显示昵称）。
    * 如有陌生人混入，你可以将其移除。
    * **救援功能**：如果有迷糊的朋友**弄丢了自己的 Key**，你可以在后台帮他查到，请私信发给他。
3. **开启抽选**：
    * 到达截止时间后，点击“开始抽选”。系统会随机生成一个“送礼循环圈”（A送给B，B送给C...）。
    * *放心，后台默认开启“防剧透模式”，你只会看到“抽选已完成”，不会直接被剧透谁送给你礼物。*
4. **揭晓（可选）**：
    * 默认情况下，大家只能看到“我要送给谁”。
    * 当你觉得时机成熟（比如大家都收到礼物后），你可以在后台开启“公开送礼人”，这样大家就能看到是谁送给自己的礼物了。

---

## 🎅 如果你是参与玩家

你的任务是报名、填写收货信息，并准备一份心意。

1. **填写报名表**：
    * 输入房主分享的 `邀请 Key`。
    * 填写你的昵称（方便大家认出你）。
    * 填写**收货信息**（姓名、电话、地址）和**祝福语**。
2. **获取你的专属凭证**：
    * 提交后，系统会给你一个 🎫 **参与 Key**。
    * **⚠️ 请务必截图或复制保存好这个 Key！** 它是你后续查看“我要送礼给谁”的唯一凭证，就像你的电影票一样。
    * *如果不小心弄丢了，请立刻私聊房主帮忙找回。*
3. **修改资料**：
    * 在房主点击“开始抽选”之前，你随时可以用 `参与 Key` 登录修改地址或昵称。一旦抽选开始，为了保证物流准确，信息将锁定不可修改。
4. **查看送礼目标**：
    * 抽选完成后，使用 `参与 Key` 登录，你将看到你要**守护（送礼）的对象**，以及TA的收货地址和心愿。
    * 此时你暂时看不到是谁送给你的（保持神秘感✨），直到房主开启揭晓模式。

---

## 🔒 关于隐私与安全（大家最关心的）

既然涉及到电话和地址，我们非常重视信息的安全：

1. **无账号机制**：不需要手机号注册，也不需要设置密码，最大程度减少个人信息留存。
2. **仅必要可见**：
    * **房主**：只能看到你的昵称和社交账号（用于确认身份），**无法**直接查看你的真实姓名、电话和详细地址。
    * **送礼人**：只有抽中送给你礼物的那个人，才能在抽选后看到你的收货信息。
3. **数据加密**：所有敏感数据（地址、电话）在数据库中均以加密密文存储。
4. **阅后即焚**：活动彻底结束（互换完成）后，服务器将执行**物理删除**，彻底清空所有人的地址和联系方式，绝不保留备份。
